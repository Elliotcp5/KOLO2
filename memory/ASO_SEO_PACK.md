## 📋 Checklist ASO (à cocher dans App Store Connect)

- [ ] 4 localisations activées : en-US, es-ES, de-DE, fr-FR
- [ ] Screenshots localisés pour chaque langue (6.5", 5.5", iPad 12.9")
- [ ] App Preview vidéo (optionnel mais +30% de conversion)
- [ ] Category : `Business` (primaire) + `Productivity` (secondaire)
- [ ] Age rating : 4+
- [ ] Content rights : tu déclares posséder les droits sur tous les contenus
- [ ] App Privacy (iOS) : déclarer collecte de données (email, nom, payment info via Stripe, usage analytics via PostHog/GA4)
- [ ] Data Safety (Google Play) : même déclaration

## 🌐 Checklist SEO (site web)

- [x] `<title>` + `<meta description>` dynamiques selon locale
- [x] `hreflang` pour EN/FR/ES/DE/IT + x-default
- [x] Canonical URL
- [x] Open Graph + Twitter Card complets
- [x] JSON-LD SoftwareApplication (aggregateRating, offers, featureList)
- [x] JSON-LD Organization (sameAs → App Store)
- [x] `robots.txt` avec exclusion de /app et /api
- [x] `sitemap.xml` multilingue
- [x] `apple-itunes-app` meta (banner iOS natif → télécharge l'app)
- [ ] Backlinks : soumettre à Product Hunt, BetaList, Reddit r/realtors, r/RealEstate
- [ ] Contenu SEO : ajouter un blog `/blog` avec articles "Best CRM for real estate 2026" en 4 langues (gros boost organique)
